# ✈️ 自助旅行規劃 Skill：換匯與預算彈性自訂分配助手

本專案旨在解決出國旅遊時最常遇到的「預算超支」與「資金分配混亂」問題。透過結構化的 AI 工作流程思維，結合互動式前端網頁工具，協助旅者根據個人的旅行風格快速、科學地分配資金。

---

## 🛠️ 第一部分：個人工作流程 Skill

### 📌 角色設定 (Role)
你是一位資深的自助旅行規劃師與財務分析助手。專長是根據旅者的預算總額，結合自訂的消費權重，提供最即時的預算分配數據。

### 🔄 工作流程 (Workflow)
| 步驟 | 階段名稱 | 具體執行內容 | 預期產出 |
| :--- | :--- | :--- | :--- |
| **1** | **匯率換算** | 根據輸入的台幣總預算，自動換算為目的地國家（如日、韓）的外幣總額。 | 外幣總可用預算 |
| **2** | **權重自訂** | 允許使用者自由調整「住宿、美食、交通、購物」四大類別的百分比。 | 自訂化預算結構比例 |
| **3** | **智慧防呆審查** | 系統自動加總分配比例，若總和不等於 100% 則即時阻斷計算並跳出警示。 | 資料完整性校驗與防呆 |
| **4** | **動態結果輸出** | 即時將換算後的各項目外幣金額以千分位格式清晰呈現。 | 結構化旅遊預算明細 |

## 💻 第二部分：作品介紹 - 旅遊預算分配計算器

### 💡 開發動機
很多人出國只知道帶多少錢，卻不知道如何精準分配。本作品利用 **HTML/JavaScript** 技術，製作了一個直覺式的網頁計算器，讓旅者可以依據自己是「小資購物團」還是「豪華住宿團」，一分鐘內調配出最符合當次行程的用錢計畫。

### 🌟 系統特色
* **即時匯率換算**：內建熱門旅遊地（日圓、韓元、泰銖、港幣）的最新參考匯率。
* **高彈性自訂比例**：不再局限於死板的固定比例，使用者可根據旅行風格自由調配四大核心開銷。
* **嚴謹防呆審查**：若輸入的比例總和不等於 100%，系統會智慧提示錯誤，展現工業管理中對數據精準度的追求。
